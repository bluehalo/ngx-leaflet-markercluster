zoomToShowLayer — Zone.js Timing

MarkerClusterGroup.zoomToShowLayer(marker, callback) zooms and pans until the target marker is visible, then fires a callback. When called inside Angular's zone, zone.js patches requestAnimationFrame and alters cluster icon rendering timing, creating a race condition where leaflet.markercluster silently drops the callback. The fix is NgZone.runOutsideAngular().

The target marker is at 40.7128, -74.0060 (NYC), clustered with 49 nearby markers. Reset the view, then try each button to compare. The broken version may not fail on every attempt — the bug is timing-dependent.
